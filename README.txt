The following files are provided as part of the one-X Communicator 6.2 distribution...

Avaya one-X Communicator Suite.exe:
Installer for the 6.2 release of one-X Communicator. It contains both Avaya one-X Communicator and the Avaya Collaboration Applications installer which provides desktop integration plug-ins for Microsoft Outlook, Microsoft Internet Explorer, Google Chrome and Apple Safari browsers.

onexcuiadmin.exe:
This is the centralized administration tool for managing the configuration of one-X Communicator for deployment. It is intended only for administrators and not end-users.
